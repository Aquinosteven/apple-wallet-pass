# ShowFi Logo Kit

## Direction

The selected direction is based on the AI-refined 11-A concept, variation A: a reversed white phone/pass mark and `ShowFi` wordmark on a dramatic full-canvas blue gradient. The mark keeps the clean phone silhouette from the earlier option 01, with a horizontal ticket/pass cutting through the middle of the phone.

## Assets

- `public/showfi-logo-stacked.svg` - primary full-gradient stacked logo.
- `public/showfi-logo-gradient.svg` - same primary full-gradient stacked logo, named explicitly for gradient usage.
- `public/showfi-mark-gradient.svg` - gradient icon-only mark.
- `public/showfi-logo.svg` - horizontal logo for headers and wider layouts on light backgrounds.
- `public/showfi-mark.svg` - white icon-only mark for gradient/dark backgrounds.
- `public/showfi-icon.svg` - favicon/app icon version with gradient background.
- `public/showfi-logo-mono.svg` - monochrome stacked version.
- `public/showfi-ai-concepts.png` - AI exploration sheet used as visual input.
- `public/logo-exports/` - PNG icon exports at 32, 64, 128, 256, 512, and 1024 px.

## Usage Notes

Use the full-gradient stacked logo when ShowFi needs a centered brand moment. Use the horizontal logo in compact navigation or footer contexts. Use the mark or favicon only when the wordmark would be too small to read. Keep clear space around the logo equal to at least the phone notch height.
