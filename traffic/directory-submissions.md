# ShowFi Directory Submission Workflow

Updated: 2026-03-20

## Scope

ShowFi.io is wallet pass software for Apple Wallet and Google Wallet. Primary use cases are booked sales calls, webinars, challenges, and live events where show rate matters. This workflow favors high-trust software directories, startup galleries, and event-tech listings over low-quality bulk spam.

## Scoring Rubric

- `Authority estimate`: search-visibility estimate on a 100-point scale, not a vendor-reported DA metric
- `Relevance`: fit for wallet passes, events, webinars, agencies, marketing ops, CRM, or SaaS
- `Dofollow`: `Yes`, `No`, `Mixed`, or `Unknown`
- `Acceptance`: estimated listing likelihood for ShowFi today
- `Traffic`: estimated referral upside from an approved listing
- `Submission`: `Free`, `Paid`, `Freemium`, `Editorial`, or `Integration-gated`
